i春秋-Web-WhoAreYou

访问网站，提示无权限访问。

Sorry. You have no permissions.

F12，发现 Cookie：Zjo1OiJ0aHJmZyI7，应该是 Base64，解码后得到 f:5:"nqzva";，rot13解密下，得到 s:5:"guest";。

改成 admin 用户，在编码回来，设置到 Cookie。

再次访问提示

Hello admin, now you can upload something you are easy to forget.

右键查看源码，看到隐藏了两个参数，应该是要上传shell到服务器。

尝试直接传一句话，失败。

用特殊字符字典跑了下，发现过滤了 < ，那普通的shell可能都不能用了。

尝试了下用数组参数绕过，结果成功。

直接访问shell，拿到 flag。
